# Trust-IT - Modern Web Solutions Website

## Project Overview
Trust-IT is a responsive and modern company landing page built using HTML, CSS, and JavaScript.  
The website is designed to showcase IT services like Web Design, Web Development, and Mobile App Development.  
It includes interactive features such as a mobile navigation menu, smooth scrolling, and a contact form with validation.

---

## Technologies Used
- HTML5  
- CSS3 
- JavaScript   
- Font Awesome Icons  

---

## Features
- Fully Responsive Design (Mobile + Desktop)
- Mobile Navigation Toggle Menu
- Smooth Scroll to Contact Section
- Services Section with Icons
- About Section with Social Links
- Contact Form Validation
- Error Message Display System
